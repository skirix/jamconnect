# JamConnect — Brief Produit

## Proposition de Valeur
Plateforme communautaire française de mise en relation entre musiciens pour jouer ensemble, avec gestion d'espaces (salles de répète, bars).

## Cibles
1. **Musiciens** amateurs/pro (18-45 ans) cherchant à jouer en groupe
2. **Propriétaires de locaux** (salles de répète, bars avec scène)

## Fonctionnalités MVP
| Priorité | Feature | Description |
|----------|---------|-------------|
| P0 | Auth | Email/password avec Supabase |
| P0 | Profil | Instruments, styles, niveau, localisation |
| P0 | Recherche | Par style + rayon géographique |
| P0 | Jam Sessions | Créer/joindre des jams (mode public) |
| P1 | Contact | Email masqué entre musiciens |
| P2 | Profil Local | Pour propriétaires d'espaces |

## Hors Scope MVP
- Marketplace matériel (V2)
- Mode "inscription obligatoire" (V2)
- Chat temps réel (V2)
- App mobile native (V2 - PWA d'abord)

## KPIs
- 100 inscrits mois 1
- 50 profils complets
- 20 jams créées
- 30% taux conversion contact

## Business Model
Gratuit communautaire. Revenus futurs : commission marketplace, freemium pour pros.
